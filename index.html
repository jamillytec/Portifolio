Desenvolvimento Web

Esta disciplina abrange diversas áreas do desenvolvimento web, desde a criação de páginas web estáticas até a criação de aplicações web dinâmicas e interativas.
O HTML é a linguagem de marcação que define a estrutura e o conteúdo de uma página web. Ele é usado para criar cabeçalhos, parágrafos, listas, imagens e links, entre outros elementos.
O CSS é a linguagem de estilo que permite a personalização visual das páginas web. Ele é usado para alterar cores, fontes, tamanhos e posicionamento de elementos HTML, criando layouts atraentes e responsivos.

Redes de computadores

O curso de Redes de Computadores é uma formação acadêmica direcionada ao estudo, projeto, implementação e gerenciamento de sistemas de redes que permitem a comunicação entre dispositivos de computação.
O programa abrange tópicos que incluem a arquitetura de redes, protocolos de comunicação, segurança de redes, administração de sistemas, além de tecnologias de rede sem fio e com fio.
Durante a formação, os alunos são introduzidos aos princípios das redes, como os modelos OSI e TCP/IP, que dizem respeito a como as informações são estruturadas, gerenciadas e transmitidas entre dispositivos.
